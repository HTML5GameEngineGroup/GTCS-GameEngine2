Behavior Controls:
  • P: Penetration resolution on/off
  • V: Motion on/off
  • H: Inject random velocity to all objects
  • G: Create new Rectangle/Circle (randomly)
  • WASD-ZX: Move (ZX-rotate) selected object
  • Left/Right: Sequence through selected object
  • Up/Down + MRF: Increase/Decrease Mass/Restitution/Friction
Drawing Controls:
  • T:Texture 
  • R:RigidShape 
  • B:Primitive Bounds (only when RigidShape shows) 
C: Generate particles at the mouse position.
